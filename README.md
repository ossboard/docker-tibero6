
## 실행방법
```
brew intall --cask docker
brew install docker-compose

폴더생성및 압축푼후,
1. 라이선스 발급후, setting/license.xml 
  (귀찮으면 hostname 을 'dummy' 로 주면, 구글에 공유된 라이선스도 동작은 하나 언제 만료될지는 모름 )

2. docker-compose up <-- 실행후, 오류없이 잘 뜨는지 볼것!
3. 이상없음 
 sh startup.sh up
 sh startup.sh start
 sh startup.sh shell
 ... 등으로 관리 가능

4. share 폴더에 스크립트등 넣어둠
5. setting폴더에 .bashrc 파일도 추가해서 콘솔 path등 추가

```
